# Aim Trainer Pro

An interactive aim training webpage that calculates reaction time and provides a rating system.

## Features

- **Reaction Time Measurement**: Tracks time from target appearance to click
- **Rating System**: Provides ratings from "Slow" to "Elite" based on performance
- **Comprehensive Stats**: Displays reaction time, accuracy, hits, and average time
- **Visual Feedback**: Colorful targets with visual hit feedback
- **History Tracking**: Keeps track of recent reaction times
- **Responsive Design**: Works on different screen sizes

## How to Use

1. Open `index.html` in any modern web browser
2. Click "Start Game" to begin
3. Click on targets as quickly as they appear
4. Your reaction time and rating will update in real-time
5. Use "Pause" to pause the game and "Reset" to start over

## Rating System

- **Elite**: <150ms with >90% accuracy
- **Very Fast**: 150-200ms with >80% accuracy
- **Fast**: 200-300ms with >70% accuracy
- **Average**: 300-500ms with >60% accuracy
- **Slow**: >500ms or <60% accuracy
